Manual Local Test Plan

  1. Run npm test, npm run typecheck, npm run lint, and npx expo-doctor.
  2. Start local Expo only when ready to manually inspect runtime behavior: npx expo start.
  3. Open iOS Simulator from Expo CLI or run npx expo start --ios.
  4. Open Settings, enter a personal OpenRouter token, choose source language, target language, model preset, and cleanup default.
  5. Record a short Transcribe clip and confirm the transcript saves to History as text only.
  6. Record a Translate clip and confirm translated output is emphasized, original text remains visible, and the result saves only after translation succeeds.
  7. Translate manual text to Spanish, Arabic, and English.
  8. Edit a saved result in Record, then confirm History preview shows the edited text.
  9. Use Copy and Share from the edited result and confirm the edited text is used.
  10. Add tags from the result card, open History, filter by the tag, then delete the item.
  11. Open History detail and verify editing, Copy, Share, add tag, remove tag, and delete all use the saved item correctly.
  12. Verify Arabic text remains readable in result, history list, and history detail views.
  13. Exercise failure cases: no token, denied microphone permission, offline/network failure, auth/payment/rate-limit/provider overload, and malformed provider response via mocked or controlled test conditions.
