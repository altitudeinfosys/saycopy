Setup guide

Connect SayCopy to OpenRouter.

SayCopy does not require its own account. You provide an OpenRouter API key so requests are billed directly to your OpenRouter account.

1

Create or sign in to OpenRouter

Visit openrouter.ai and sign in. OpenRouter is a separate service with its own terms and privacy policy.

2

Add credits and set a limit

Add a small amount of credit, then set a spending limit that fits your use. SayCopy itself is free, but the models you choose may charge for usage through OpenRouter.

3

Create an API key

Open OpenRouter Keys, create a key for SayCopy, and copy it. A separate key makes it easy to monitor or revoke access later.

4

Save it in SayCopy

In the SayCopy app, open Settings, paste your key into the OpenRouter token field, and tap Save token. The key is stored in secure iOS storage on your device.

5

Record, then copy or share

Return to Record, choose Transcribe or Translate, and speak. When the result is ready, copy it or share it into another app.

Keep your key private

Treat an API key like a password. Do not post it, email it, or share screenshots that show it. If you think it has been exposed, revoke it in OpenRouter and create a new one.

Need help? Use the SayCopy contact form without including your API key.
